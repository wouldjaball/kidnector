# Kidnector

## MVP Overview
Kids earn screen time through daily affirmations.

## Setup
1. Clone repo
2. `npm install`
3. Set up .env with Supabase credentials
4. `npm start`

## Tech Stack
- React Native (Expo)
- Supabase
- RevenueCat

## Launch Timeline
- Week 1 (Current): MVP Foundation
- Week 2: Beta Testing
- Week 3: Landing Page
- Week 4: Public Launch

## Key Features
- Parent/child account creation
- Daily affirmation recording
- Screen time tracking
- Streak rewards
